# Women Safety Smart Device

## Overview
Women Safety Smart Device is a web-based application developed to improve women's safety by providing smart emergency features. The application enables secure registration, user verification, emergency contact management, GPS-based location tracking, and SOS alert functionality.

## Features
- User Registration
- Login & Authentication
- OTP Verification
- Aadhaar/Government ID Verification
- Emergency Contact Management (4–5 Contacts)
- Blood Group & Medical Information
- GPS-Based Home Address
- SOS Alert System
- Location Tracking
- Safe PIN
- Security Question
- Privacy Policy
- Terms & Conditions
- Responsive and User-Friendly Interface

## Technology Stack
- React.js
- TypeScript
- Vite
- HTML
- CSS

## Project Flow
1. Home Page
2. User Registration
3. Password Setup
4. Identity Verification
5. Emergency & Safety Information
6. Dashboard
7. Settings
8. SOS Features

## Objective
To provide a secure, reliable, and user-friendly platform that helps women respond quickly during emergencies through smart safety features.
